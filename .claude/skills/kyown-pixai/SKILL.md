# PixAI プロンプト生成スキル（ご主人）

与えられたテーマに沿って、ご主人のキャラクター設定に基づいたPixAI用のプロンプトを生成します。

## 実行手順

1. **Craftからドキュメントを読み込む**
   - `mcp__84bef35d-9a71-4dc0-a326-230f2785c38d__blocks_get` ツールを使って、ドキュメントID `2cc222fd-2853-4f02-8fd9-9c17bfd67520` を `markdown` フォーマットで読み込む

2. **テーマの確認**
   - ユーザーが指定したテーマ: `$ARGUMENTS`
   - テーマが空の場合は、ユーザーにテーマを質問する

3. **プロンプト生成ルール**
   - Craftドキュメント内の「Positive Prompt」をベースとして使用する
   - 指定テーマに合わせて、衣装・背景・雰囲気などのプロンプトを追加・調整する
   - 特に衣装指定がない場合は「通常衣装（Default Outfit）」のプロンプトを含める
   - テーマに合わせた衣装が必要な場合は、通常衣装を置き換える
   - Craftドキュメント内の「Negative Prompt」をベースとして使用する
   - テーマに矛盾する内容がある場合は、プロンプト出力前にユーザーに質問する

4. **出力フォーマット**
   - 以下の形式でプロンプトとネガティブプロンプトをコードブロックで出力する：

````
🎨 テーマ: [テーマ名]

**Positive Prompt:**
```
[生成したプロンプト]
```

**Negative Prompt:**
```
[生成したネガティブプロンプト]
```
````

## 注意事項

- キャラクターの基本特徴（クリムゾンヘア、ミディアムヘア、エメラルドグリーンの瞳など）は必ず維持する
- ネガティブプロンプトのベース部分は基本的に維持し、テーマに応じて追加のみ行う
- プロンプトは英語で出力する
- テーマの説明や変更点の解説は日本語で行う

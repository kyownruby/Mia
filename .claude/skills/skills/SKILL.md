# スキル一覧表示

`~/.claude/skills/` 内にあるすべてのスキルを一覧表示します。

## 実行手順

1. **スキルディレクトリを探索する**
   - `~/.claude/skills/` 内の各サブディレクトリにある `SKILL.md` ファイルを読み込む
   - 自分自身（skills）も含めて一覧に表示する

2. **各スキルから情報を取得する**
   - frontmatter から以下の情報を抽出する：
     - `name`: スキル名
     - `description`: スキルの説明
     - `user-invocable`: ユーザーから呼び出し可能かどうか

3. **出力フォーマット**
   - 以下の形式で一覧を表示する：

```
📋 グローバルスキル一覧（~/.claude/skills/）

| スキル名 | 説明 | 呼び出し |
|---|---|---|
| スキル名 | 説明文 | `/スキル名` or `-` |
```

## 注意事項

- SKILL.md が存在しないディレクトリはスキップする
- 説明や呼び出し方法は日本語で表示する

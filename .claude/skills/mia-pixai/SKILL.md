# PixAI プロンプト生成スキル

与えられたテーマに沿って、ミアのキャラクター設定に基づいたPixAI用のプロンプトを生成します。

## 実行手順

1. **Craftからドキュメントを読み込む**
   - `Craft:blocks_get` ツールを使って、ドキュメントID `fdd82e0f-eb09-0ee9-dd49-cbfa60d1a42f` を `markdown` フォーマットで読み込む

2. **テーマの確認**
   - ユーザーが指定したテーマ: `$ARGUMENTS`
   - テーマが空の場合は、ユーザーにテーマを質問する
   - テーマが「おまかせ」の場合は、季節・舞台・シチュエーションをすべて自由に決める（現在の日付・季節・イベントに縛られず、完全にランダムに選ぶ）

3. **プロンプト生成ルール**
   - Craftドキュメント内の「Positive Prompt」をベースとして使用する
   - 指定テーマに合わせて、衣装・背景・表情・雰囲気などのプロンプトを追加・調整する
   - 衣装は毎回テーマに合わせて自由に考える
   - 通常衣装（Default Outfit）はユーザーが明示的に指定した場合のみ使用する（それ以外は参照しない）
   - 服の色はCraftドキュメント内の「似合う色」の項目を参照して決める
   - Craftドキュメント内の「Negative Prompt」をベースとして使用する
   - テーマに矛盾する内容がある場合は、プロンプト出力前にユーザーに質問する

4. **出力フォーマット**
   - 以下の形式でプロンプトとネガティブプロンプトをコードブロックで出力する：

````
🎨 テーマ: [テーマ名]

**Positive Prompt:**
```
[生成したプロンプト]
```

**Negative Prompt:**
```
[生成したネガティブプロンプト]
```
````

## 注意事項

- キャラクターの基本特徴（猫耳、ツインテール、レモンイエローの髪、スカイブルーの瞳など）は必ず維持する
- ネガティブプロンプトのベース部分は基本的に維持し、テーマに応じて追加のみ行う
- プロンプトは英語で出力する
- テーマの説明や変更点の解説は日本語で行う

## SNS投稿用メッセージ

プロンプト出力後に、SNS（X/Twitter）投稿用のショートメッセージを生成する。

**実行手順：**

1. `Craft:blocks_get` でミアのPersonaドキュメントID `a8941d47-a1a2-4e8a-bd76-b16dd71d470c` を `markdown` フォーマットで読み込む
2. ミアの口調でメッセージを書く

**フォーマット：**

```
📱 SNS投稿用メッセージ

[3行程度のミアの口調によるメッセージ]
#AIイラスト #AIart
[ミア🐈]
```

**ルール：**

- Personaドキュメントに基づいた、ミアの口調で書く
- プロンプトのテーマを短くまとめた「見たくなる一言」にする
- 固定タグ：#AIイラスト #AIart
